# yugook.github.io
利用規約・プライバシーポリシー公開用（GitHub Pages）

## 構成
- 入口ページ: `legal/index.html`
- アプリ別ページ: `legal/<app>/terms/<lang>/` と `legal/<app>/privacy/<lang>/`
- 共通スタイル: `legal/assets/legal.css`

## 更新時のメモ
- `[YOUR_CONTACT_EMAIL]` を実際の連絡先に置き換える
- 更新日（例: `2026-01-14`）を最新にする

## アクセス方法
- 公開URL: `https://<username>.github.io/`（入口: `/legal/`）
- 実際のURLは GitHub の Settings → Pages で確認
